# Rolagem-de-dados
Projeto criado 100% por IA (Gemini 3) para fins de estudo, apenas fiz as exigências e tudo foi realizado pelo Gemini.

O projeto foi feito em Python e JavaScript, usei Flask, Flask-SocketIO, HTML5, CSS3.
Tudo foi feito apenas para uso pessoal e desenvolvimento intelectual, é possível utilizar o site em computadores diferentes utilizando um serviço de VPN, pedi ajuda a um amigo para testar, e funcionou perfeitamente quando estávamos na mesma reded (eu havia hospedado o servidor, e ele entrou).
